# Momus.io API

## Purpose

API serving the Momus.io web app

`https://api.momus.io/{route}`

## Stack

### Server

- Node
- Express
- Deployed via Heroku

### Database

- PostgreSQL
- Prisma ORM
- Deployed via ElephantSQL

## Routes/Tables

- Authors
- Works
- Deities
- Items
- Creatures
- Mortals
